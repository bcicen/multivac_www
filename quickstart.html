Quickstart

To get started with Multivac using Docker, see the Multivac README

Installing

Multivac requires Python3 
git clone https://github.com/bcicen/multivac.git
cd multivac
pip3 install -r requirements.txt
python3 setup.py install

Configuration

Copy the sample_config.yml file or create your own config file as described here

Running

To bring up a full stack, start the worker, api, and slackbot processes using the Multivac command line utility:

multivac -c /path/to/config.yml worker
multivac -c /path/to/config.yml api
multivac -c /path/to/config.yml slackbot

Usage: 

usage: multivac [-h] [-c CONFIG_PATH] [-d] {worker,slackbot,api}

multivac v0.7

positional arguments:
  {worker,slackbot,api}

optional arguments:
  -h, --help            show this help message and exit
  -c CONFIG_PATH        path to config file (default: /etc/multivac.yml)
  -d                    enable debug output
