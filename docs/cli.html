Commandline Reference

After installing Multivac, components can be launched with the cli utility:

multivac [-h] [-c CONFIG_PATH] [-d] {worker,slackbot,api,console}

Arguments

  • -c

    • Path to multivac config file (default: /etc/multivac.yml)

  • -d

    • Run in debug mode

  • -h

    • Show help dialog and exit

  • role

    • One of worker, slackbot, api or console
