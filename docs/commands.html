Commands

Multivac comes with a number of built-in commands and allows any arbitrary script or executable to be added as a custom command.

BuiltIn Commands

  • jobs <all,pending,completed,ready>

    • Returns information on jobs in a given state.

  • logs <id>

    • Returns the output log for a given job ID

  • confirm <id>

    • Confirms a pending job, given a job ID

  • help

    • Returns a list of all builtin commands and configured action commands

  • workers

    • Returns currently registered workers by name and hostname

Action Commands

Action commands are configured commands added to Multivac.

To add a new command, append it to the "actions" section of your Multivac configuration file
